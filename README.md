# Netflix-Data-Analysis

## Project Overview

This project demonstrates an end-to-end **Data Analysis workflow** using industry-standard tools. The goal is to extract meaningful insights from a Netflix dataset by performing data cleaning, transformation, querying, and visualization.

The project follows a structured pipeline:
**Python → SQL → Power BI → Presentation**


## Tech Stack

* **Python** (Pandas, NumPy) – Data Cleaning & Preprocessing
* **MySQL** – Data Storage & Querying
* **Power BI** – Data Visualization & Dashboarding


  
## Project Workflow

### 1) Data Collection

* Imported raw dataset (CSV format) into Python environment

### 2) Data Cleaning & Preprocessing (Python)

* Handled missing values
* Removed duplicate records
* Converted data types
* Transformed columns (e.g., list → string)
* Dropped unnecessary columns

### 3️) Data Storage (SQL)

* Created database and tables in MySQL
* Loaded cleaned dataset into SQL using SQLAlchemy
* Ensured proper schema and data types

### 4️) Data Analysis (SQL Queries)

Performed analytical queries such as:

* Total number of movies and TV shows
* Content distribution by genre
* Year-wise release trends
* Country-wise content distribution
* Duration analysis

### 5️) Data Visualization (Power BI)

* Connected MySQL database to Power BI
* Built an interactive dashboard with:

  * 📊 Genre distribution charts
  * 📈 Year-wise trend analysis
  * 📍 Country-based insights
  * 🎯 KPI cards (Total Movies, TV Shows, etc.)

### 6️) Presentation

* Created a presentation summarizing:

  * Key insights
  * Business understanding
  * Visual storytelling
This project is created for learning and portfolio purposes to demonstrate practical data analysis skills.
